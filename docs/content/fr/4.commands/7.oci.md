# dnx oci

```bash
dnx oci <command> [options]
```

## Sous-commandes

```bash
dnx oci build <workload>   # Build l'image
dnx oci push <workload>    # Push vers le registry
dnx oci pull <workload>    # Pull sur un serveur
dnx oci images <server>    # Lister les conteneurs
dnx oci cleanup <server>   # Nettoyer
```

## Exemples

```bash
# Build avec tag custom
dnx oci build web --tag v1.2.3 --platform linux/amd64

# Push vers un registry spécifique
dnx oci push web --registry ghcr.io/myorg/web --username myuser

# Pull sur le serveur prod-1
dnx oci pull web prod-1

# Lister les conteneurs qui tournent
dnx oci images prod-1

# Nettoyer les anciennes images (garder les 5 dernières)
dnx oci cleanup prod-1 --keep 5
```
