# Commandes DNX

## Commandes globales

```bash
dnx help [command]     # Aide détaillée
dnx version            # Version du CLI
```

## Configuration

```bash
dnx init               # Initialise un projet (crée dnx.yaml)
dnx config validate    # Valide le dnx.yaml
dnx config show        # Affiche la configuration
dnx config diff        # Diff entre deux environnements
```

## Build & Deploy

```bash
dnx build <workload>   # Build le workload
dnx deploy <workload>  # Déploie sur les serveurs
```

## Workloads

```bash
dnx workload start|stop|restart|status|logs|remove <name>
dnx status             # Statut de tous les workloads
```

## Proxy (Caddy)

```bash
dnx proxy start|restart|stop|status|logs
dnx proxy route list|add|remove
dnx proxy ssl
```

## Secrets

```bash
dnx secrets init|set|extract|print|remove
```

## Rollback

```bash
dnx rollback <workload> [--tag <tag>]
```

## Health

```bash
dnx health check [workload]
dnx health watch [workload]
dnx health history [--workload <name>]
```

## Agent & OCI

```bash
dnx agent install|uninstall|status <server>
dnx oci build|push|pull|images|cleanup
```

## Locks

```bash
dnx lock list
dnx lock release <resource>
```

## Options globales

Toutes les commandes supportent :

| Flag | Description |
|------|-------------|
| `--verbose, -v` | Mode verbeux |
| `--quiet, -q` | Mode silencieux |
| `--json` | Sortie JSON |
| `--help, -h` | Aide |
