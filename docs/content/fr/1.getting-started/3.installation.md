# Installation

## Prérequis

- **Bun** ≥ 1.0 (`curl -fsSL https://bun.sh/install | bash`)
- **Git**
- Optionnel : **flox** ([flox.dev](https://flox.dev)) pour les environnements reproductibles
- Optionnel : **Docker** ou **Podman** pour le runtime OCI

## Installer DNX

```bash
# Via npm
npm install -g @dnax/dnx

# Via Bun (recommandé)
bun install -g @dnax/dnx
```

Vérifiez l'installation :

```bash
dnx version
# dnx v0.1.0 — Bun 1.3.14
```

## Vérifier les dépendances optionnelles

```bash
# Pour le runtime flox
flox --version

# Pour le runtime Docker/OCI
docker --version
# ou
podman --version

# BuildKit standalone (optionnel, pour build OCI sans Docker)
buildctl --version
```

## Mise à jour

```bash
bun update -g @dnax/dnx
```
