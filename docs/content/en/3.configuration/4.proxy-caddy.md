# Proxy & Caddy

DNX integrates Caddy as a reverse proxy and load balancer. Configuration is auto-generated from `dnx.yaml`.

## Configuration

```yaml
proxy:
  auto_ssl: true
  email: admin@example.com
  routes:
    - domain: app.example.com
      target: web
      port: 3000
      lb_policy: round_robin
      ssl: true
      headers:
        X-Frame-Options: DENY
```

## Commands

```bash
dnx proxy start --env staging
dnx proxy restart
dnx proxy stop
dnx proxy status --env production
dnx proxy logs server-name --follow
```

## Load Balancing

Policies: `round_robin`, `least_conn`, `first`, `ip_hash`.

## SSL

Caddy auto-manages SSL certificates via Let's Encrypt. No additional configuration needed.

```bash
dnx proxy ssl
```

## Rate Limiting

```yaml
proxy:
  global_options:
    rate_limit:
      rate: 100
      burst: 50
      period: 1s
```
