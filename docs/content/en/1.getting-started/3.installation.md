# Installation

## Prerequisites

- **Bun** ≥ 1.0 (`curl -fsSL https://bun.sh/install | bash`)
- **Git**
- Optional: **flox** ([flox.dev](https://flox.dev)) for reproducible environments
- Optional: **Docker** or **Podman** for OCI runtime

## Install DNX

```bash
# Via npm
npm install -g @dnax/dnx

# Via Bun (recommended)
bun install -g @dnax/dnx
```

Verify installation:

```bash
dnx version
# dnx v0.1.0 — Bun 1.3.14
```

## Verify optional dependencies

```bash
# For flox runtime
flox --version

# For Docker/OCI runtime
docker --version
# or
podman --version

# BuildKit standalone (optional, for OCI builds without Docker)
buildctl --version
```

## Upgrade

```bash
bun update -g @dnax/dnx
```
