# DNX Commands

## Global

```bash
dnx help [command]
dnx version
```

## Configuration

```bash
dnx init
dnx config validate|show|diff
```

## Build & Deploy

```bash
dnx build <workload>
dnx deploy <workload>
```

## Workloads

```bash
dnx workload start|stop|restart|status|logs|remove <name>
```

## Proxy (Caddy)

```bash
dnx proxy start|restart|stop|status|logs
dnx proxy route list|add|remove
dnx proxy ssl
```

## Secrets

```bash
dnx secrets init|set|extract|print|remove
```

## Rollback

```bash
dnx rollback --workload <name> [--version]
dnx rollback history --workload <name>
```

## Health

```bash
dnx health check [workload]
dnx health watch [workload]
dnx health history
```

## Agent & OCI

```bash
dnx agent install|uninstall|status <server>
dnx oci build|push|pull|images|cleanup
```

## Locks

```bash
dnx lock list
dnx lock release <resource>
```

## Global Options

| Flag | Description |
|------|-------------|
| `--verbose, -v` | Verbose mode |
| `--quiet, -q` | Quiet mode |
| `--json` | JSON output |
| `--help, -h` | Help |
